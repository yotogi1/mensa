# MENSA対策 推理トレーニング Pro

## 使い方
1. zipを解凍
2. `index.html` をブラウザで開く
3. スマホの場合はGitHub Pagesに置くとアプリ風に使えます

## v2で改善したこと
- 問題数を増加
- 解説を詳細化
  - 答えの要点
  - 見つけ方
  - 計算・判断
  - 本番での考え方
- 分野別演習を追加
- 復習リストを追加
- 間違えた問題を自動保存
- 成績分析に平均秒数を追加
- 前版の曖昧問題・解答不整合を修正

## 注意
実際のMENSA入会テスト問題を再現したものではありません。
合法的な推理力・処理速度トレーニング用です。
