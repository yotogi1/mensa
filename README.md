# 図形特化 MENSAトレーナー

## 内容
図形問題が苦手な方向けに、以下だけを集中練習できるHTMLアプリです。

- 回転
- 反転
- 回転・反転複合
- 図形行列
- 規則発見

## 収録問題数
48問

## 使い方
1. zipを解凍
2. `index.html` をブラウザで開く
3. スマホで使う場合はGitHub Pagesに置くと便利です

## 機能
- 分野別演習
- 基礎10問モード
- 弱点集中
- 復習リスト
- 成績分析
- 図形の見方講座
