# MENSA対策 推理トレーニング

## 使い方
1. zipを解凍
2. `index.html` をブラウザで開く
3. スマホの場合はGitHub Pages等に置くとアプリ風に使えます

## 内容
- 数列推理
- 図形行列
- 論理推理
- 類推
- 空間認識
- 処理速度
- 制限時間トレーニング
- 弱点分析
- ローカル保存

## 注意
実際のMENSA入会テスト問題を再現したものではありません。
合法的な推理力・処理速度トレーニング用です。
